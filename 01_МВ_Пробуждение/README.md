# Книга 1 — «Мирный Воин. Пробуждение»

Лид-магнит → **МВ Тренинг** (3-дневный тренинг «Мирный Воин»)

## Структура

- **`тексты/`** — MD-исходники. Редактировать здесь, затем регенерировать PDF.
- **`pdf/`** — Финальные PDF для раздачи. Не редактировать вручную.
- **`скрипты_pdf/`** — Python-скрипты (ReportLab). Эталонный: `generate_pdf_yama.py`.
- **`маркетинг/`** — Базы сегментов, голоса, болей, мастер-промт.

## Регенерация PDF

```bash
cd скрипты_pdf/
python3 generate_pdf_СЕГМЕНТ.py
# PDF появляется в ../pdf/
```

## Важно

- Символ `→` (U+2192) не поддерживается шрифтом Lora-Italic → заменять на `->` в MD перед генерацией
- Шрифты кэшируются в `/tmp/fonts_mv/` — скачиваются автоматически при первом запуске
- Эталонный скрипт: `generate_pdf_yama.py` — копировать его при создании нового сегмента
