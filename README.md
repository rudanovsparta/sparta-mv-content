## Что это

Единая база всего контента, скриптов и маркетинговых материалов проекта **СПАРТА | Мирный Воин**.  
Сюда попадают изменения из Perplexity Computer, Cursor, Codex и любых других инструментов — в одном месте видно кто что делал, когда и зачем.

## Как работать с репозиторием

### Из Cursor / VS Code
```bash
git clone https://github.com/rudanovsparta/sparta-mv-content.git
cd sparta-mv-content
```

### Стандарт коммитов
```
[TOOL] [РАЗДЕЛ] Краткое описание

Примеры:
[Perplexity] [МВ_Пробуждение] Добавлена книга ПУСТОТА — текст + PDF
[Cursor] [Воронка] Написана Telegram-цепочка для СТЕНА
[Codex] [Скрипты] Рефакторинг generate_pdf_yama.py
[Ручной] [Бренд] Обновлён брендбук v1.3
```

### Ветки
- `main` — финальные версии
- `draft/[название]` — черновики в работе
- `tool/perplexity`, `tool/cursor`, `tool/codex` — по инструменту (опционально)
