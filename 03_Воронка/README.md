# Воронка распространения

```
САЙТ (spartamv.ru)
  ↓
КВИЗ Marquiz (анкета №1)
  ↓
СЕГМЕНТАЦИЯ (5 вариантов: СТЕНА / ЯМА / ПЛАТО / ПУСТОТА / НЕ УВЕРЕН)
  ↓
СТРАНИЦА РЕЗУЛЬТАТА (персонализированная под сегмент)
  ↓
КНИГА PDF — Глава 1 выдаётся сразу
  ↓
TELEGRAM-ЦЕПОЧКА — Главы 2–7 по одной в день (7 дней)
  ↓
ЗВОНОК (квалификация)
  ↓
МВ ТРЕНИНГ (3 дня)
  ↓
ПМВ (Путь Мирного Воина)
```

## Статус компонентов

| Компонент | Статус |
|-----------|--------|
| Квиз Marquiz | Существует (настраивается на стороне Marquiz) |
| Страницы результата | ❌ Тексты не написаны |
| PDF книги (Гл.1) | ✅ Все 5 сегментов готовы |
| Telegram-цепочка (Гл.2–7) | ❌ Не написана |
| Звонок (скрипт) | Не в этом репо |

## Папки

- **`marquiz_страницы_результата/`** — тексты страниц после прохождения квиза
- **`telegram_цепочка/`** — сообщения для рассылки глав 2–7 (по сегментам)
