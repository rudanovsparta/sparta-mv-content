# Страницы результата Marquiz

## Статус: ❌ Не написаны

## Что это

Страница, которую человек видит СРАЗУ после прохождения квиза — до получения PDF.

### Структура каждой страницы
1. **Заголовок** — название сегмента («Ты в режиме СТЕНЫ»)
2. **Описание состояния** — 3-4 предложения, читатель узнаёт себя
3. **Что внутри книги** — без спойлеров, интрига
4. **CTA-кнопка** — «Получить первую главу» → PDF
5. **Мини-preview** — первый абзац из Главы 1

### Нужно для сегментов
- [ ] СТЕНА (48%)
- [ ] ЯМА (8%)
- [ ] ПЛАТО (37%)
- [ ] ПУСТОТА (7%)
- [ ] НЕ УВЕРЕН (смешанный)
